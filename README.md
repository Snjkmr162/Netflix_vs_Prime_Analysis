# 🎬 Netflix vs Prime Video — Content Strategy Analysis

> A full-stack data science project comparing Netflix and Amazon Prime Video
> using live API data, quantitative analysis, and an interactive Streamlit dashboard.

![Dashboard](visuals/00_summary_dashboard.png)

---

## 🧠 Project Thesis

> *"Is Amazon Prime Video declining compared to Netflix?"*

This project answers that question with data — pulling **live catalogs** from
Watchmode and TMDb APIs, engineering features, and surfacing insights through
a Streamlit dashboard.

**Part 2** (qualitative sentiment analysis via Reddit) is in progress.

---

## 🔍 Key Findings

| Metric | Netflix | Prime Video | Winner |
|---|---|---|---|
| Avg TMDb Rating | 7.49 | 7.40 | Netflix |
| Avg Popularity Score | 18.1 | 19.5 | Prime Video |
| International Content | 19.2% | 6.0% | Netflix |
| Animation Titles | 29 | 15 | Netflix |
| Drama Titles | 72 | 86 | Prime Video |
| 2020s Content Share | 83% | 57% | Netflix |

**Headline insight:** Netflix is investing 3x more in international content
and shows a widening quality-trend gap vs Prime Video since 2022 — despite
both platforms having statistically comparable overall ratings (p = 0.12).
